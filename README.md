# Optical
**Real-Time AI That Explains the World Around You**

Optical is an AI-powered visual assistant that uses your camera and voice to understand objects in the real world and explain how they work. Point your camera at something, ask a question, and Optical will identify the object and teach you about it in real time.

Think of it as **Shazam for the physical world**.

---

## 🚀 Features

- 📷 **Live Camera Understanding**  
  Detect and interpret objects directly from your camera feed.

- 🎤 **Voice Interaction**  
  Ask questions naturally using your microphone.

- 🧠 **Real-Time Explanations**  
  Optical explains what you're seeing and how it works.

- 🎓 **Learning Mode**  
  Get simplified explanations and quizzes to reinforce understanding.

- ⚡ **Low Latency AI Responses**  
  Powered by real-time multimodal AI for fast interaction.

---

## 🧩 Example Use Cases

- 🔧 Learn how mechanical systems work (bikes, engines, tools)
- ☕ Understand everyday devices (coffee machines, appliances)
- 🧑‍🎓 Study diagrams, lab setups, or engineering components
- 🏋️ Learn how gym equipment functions
- 🧠 Turn the real world into an interactive classroom

## 🏗️ Tech Stack

**Frontend**
- Next.js
- React
- TailwindCSS
- WebRTC / `getUserMedia` for camera and microphone

**Backend**
- Node.js
- Express / Fastify
- WebSockets for real-time streaming

**AI**
- Gemini Live API (multimodal voice + vision)
